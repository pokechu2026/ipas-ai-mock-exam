# Notion 嵌入建議

## 建議嵌入哪個網址

- 學員入口：`https://<你的帳號或組織>.github.io/<repo>/`
- 直接考試：`https://<你的帳號或組織>.github.io/<repo>/exam.html`
- 老師解答版：`https://<你的帳號或組織>.github.io/<repo>/review.html?mode=answer-key`

## 在 Notion 裡怎麼放

1. 打開你的 Notion 頁面
2. 輸入 `/embed`
3. 貼上 GitHub Pages 網址
4. 調整嵌入區塊高度，建議至少 900px

## 教學使用建議

- 課前練習頁：嵌入首頁，讓學生自己選正式考試版或解答版
- 正式測驗頁：嵌入 `exam.html`，學生作答後會自動跳到解析頁
- 老師講解頁：嵌入 `review.html?mode=answer-key`

## 提醒

- 如果 Notion 快取舊畫面，可在網址後加上版本參數，例如 `?v=20260317`
- GitHub Pages 首次發佈後通常要等幾分鐘網址才會生效
